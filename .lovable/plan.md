## סיכום

עדכון מקיף של כל התוכן והקריאייטיב באתר כך שישקף את שני השירותים העיקריים: **"הדרך לדירה"** (הכשרה דיגיטלית) ו**"ליווי קרנף פרימיום"** (ליווי אישי מקצה לקצה). בנוסף - חיבור טפסים לשליחת הודעות WhatsApp/מייל, והוספת קישורים לרשתות החברתיות.

## שלב 1: עדכון סקציית Hero - מיקוד המסר

**קובץ:** `Hero.tsx`

- שינוי הכותרת הראשית לשקף את השירותים: "להרוויח כסף בנדל"ן זה לא מזל, זה מדע מדויק."
- עדכון תת-הכותרת: "קרנף נדל"ן מלווים משקיעים לרכישת דירה עם שיטה מבוססת נתונים. בין אם אתם רוכשים דירה ראשונה או משקיעים מנוסים - יש לנו את הפתרון בדיוק בשבילכם."
- שינוי כפתורי CTA: "לתוכנית הדרך לדירה" + "לליווי קרנף פרימיום"
- עדכון הטופס: שינוי ה-dropdown לשני השירותים (הדרך לדירה / ליווי קרנף פרימיום / שניהם)
- עדכון מספר WhatsApp בטופס

## שלב 7: אינטגרציית טפסים - שליחת לידים

**דרישה:** כשמישהו משאיר פרטים בטופס, לשלוח הודעה ל-WhatsApp ולמייל.

**גישה מומלצת:** שימוש ב-Lovable Cloud (Supabase Edge Function) לשליחת המייל, ופתיחת WhatsApp API ישירות מהדפדפן כ-fallback.

**קבצים חדשים:**
- `supabase/functions/send-lead/index.ts` - Edge Function ששולח מייל באמצעות Resend API ל-`karnaf.yazamut@gmail.com`

**קבצים מעודכנים:**
- `Hero.tsx` - חיבור הטופס לפונקציה + הודעת הצלחה
- `Footer.tsx` - חיבור הטופס לפונקציה + הודעת הצלחה

**תהליך הטופס:**
1. המשתמש ממלא שם, טלפון, אימייל, ובוחר שירות
2. בלחיצה על "שלחו לי פרטים":
   a. נשלח מייל עם פרטי הליד ל-karnaf.yazamut@gmail.com
   b. נפתח קישור WhatsApp עם הודעה מעוצבת מראש
3. הטופס מציג הודעת הצלחה

**הערה חשובה:** לשליחת מיילים אוטומטית נדרש חיבור Lovable Cloud + שירות מיילים (Resend). אם לא מחוברים - הטופס יפתח WhatsApp ישירות עם הפרטים.



## תוכנית: העברת "הדרך לדירה" למצב "בקרוב" + טופס רשימת המתנה

### סיכום
סריקה יסודית של הקוד מצאה **14 קבצים** שמזכירים את "הדרך לדירה". התוכנית תעדכן את כל האתר כך שהתוכנית תוצג כ"בקרוב" עם אפשרות להשאיר פרטים לרשימת המתנה.

### עמוד הקורס (CoursePage) — גרסת "בקרוב"
- **Hero**: ישאר עם הכותרת "הדרך לדירה" + badge "בקרוב!" במקום "50+ שיעורים..."
- **Trust Bar**: ישאר (מציג את מה שיהיה בתוכנית)
- **Problem/Solution**: ישאר (בונה ערך)
- **Program Cards**: ישארו (הצצה)
- **Pricing section**: הסרת מחיר וכפתורי רכישה → טופס waitlist עם שדות שם, טלפון, אימייל + כפתור "שמרו לי מקום!"
- **Quiz**: ישאר (אינטראקטיבי ומעניין)
- **Testimonials**: ישארו
- **FAQ**: עדכון תשובות רלוונטיות

### טופס רשימת המתנה (Waitlist)
- שדות: שם, טלפון, אימייל (חובה)
- שליחה דרך submit-lead עם `service: "waitlist"` ו-`source: "course-waitlist"`
- הודעת הצלחה: "נרשמת בהצלחה! נעדכן אותך ראשון/ה כשהתוכנית יוצאת לאוויר 🎉"

### עדכון Edge Function
- הוספת `"waitlist"` ל-`ALLOWED_SERVICES`
- הוספת `"course-waitlist"` ל-`ALLOWED_SOURCES`
- הוספת label: `waitlist → "רשימת המתנה — הדרך לדירה"`

### מה לא ישתנה
- עדויות (לגיטימיות, מתייחסות לתוכן שכבר נלמד)
- סילבוס ותוכן התוכנית (בונה ערך ומעורר עניין)
- עמודי בלוג, אודות, צור קשר

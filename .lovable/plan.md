

# תוסף נגישות צף — עמידה בתקן ישראלי IS 5568 ו-WCAG 2.1 AA

## רקע רגולטורי

חוק שוויון זכויות לאנשים עם מוגבלויות (1998) מחייב הנגשת אתרי אינטרנט בישראל. התקן הישראלי IS 5568 מבוסס על WCAG 2.1 רמה AA. הדרישות כוללות:

- **הצהרת נגישות** — חובה להציג באתר
- **כפתור נגישות צף** — גלוי בכל עמוד
- **התאמות טקסט** — הגדלה/הקטנה, מרווחי שורות, גופן קריא
- **התאמות ניגודיות** — ניגודיות גבוהה, גווני אפור, היפוך צבעים
- **סמן מוגדל** — עבור משתמשים עם לקויות מוטוריות
- **הדגשת קישורים** — סימון ויזואלי של כל הקישורים
- **עצירת אנימציות** — למשתמשים עם רגישות לתנועה
- **ניווט מקלדת** — תמיכה מלאה בטאב ופוקוס
- **פרופילים מוכנים** — כגון: עיוורון צבעים, ADHD, אפילפסיה

## מה ייבנה

### 1. רכיב `AccessibilityWidget` — כפתור צף + תפריט

- כפתור ♿ קבוע בצד שמאל למטה (WhatsApp בצד ימין)
- לחיצה פותחת פאנל עם כל אפשרויות ההנגשה
- תפריט בעברית, RTL מלא

### 2. אפשרויות שיוצגו בתפריט

| קטגוריה | אפשרויות |
|----------|----------|
| תוכן | הגדלת טקסט, הקטנת טקסט, מרווח שורות, גופן קריא (דיסלקסיה) |
| צבעים | ניגודיות גבוהה, היפוך צבעים, גווני אפור |
| ניווט | הדגשת קישורים, סמן גדול, מדריך מקלדת |
| תנועה | עצירת אנימציות |
| פרופילים | עיוורון צבעים, ADHD, אפילפסיה (עצירת הבהובים) |

### 3. הצהרת נגישות

- דף או מודאל עם הצהרת נגישות מלאה בעברית
- כולל: תאריך עדכון, פרטי קשר לרכז נגישות, תקן נתמך

### 4. שמירת העדפות

- שמירה ב-localStorage כדי שהעדפות ישמרו בין ביקורים

## קבצים שייווצרו / יעודכנו

- **`src/components/AccessibilityWidget.tsx`** — הרכיב הראשי (כפתור + פאנל)
- **`src/components/AccessibilityStatement.tsx`** — הצהרת נגישות
- **`src/layouts/SharedLayout.tsx`** — הוספת הרכיב ל-layout
- **`src/index.css`** — כיתות CSS להנגשה (high-contrast, grayscale, large-cursor, readable-font וכו׳)
